# Built-in extensions

Built-in extensions are maintained in Griffe's code base. They generally bring support for core features of the Python language or its standard library.

| Extension                                                                                                 | Description                                     |
| --------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| [`dataclasses`](https://mkdocstrings.github.io/griffe/extensions/built-in/dataclasses/index.md)           | Support for dataclasses.                        |
| [`unpack_typeddict`](https://mkdocstrings.github.io/griffe/extensions/built-in/unpack-typeddict/index.md) | Support for typing.Unpack and typing.TypedDict. |
