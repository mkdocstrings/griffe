# Official extensions

Official extensions are developed and maintained within the mkdocstrings organization on GitHub, in separate repositories. They generally bring support for various third-party libraries or other documentation-related features that are part of Python's standard library.

| Extension                                                                                                                 | Description                                                                                                                   |
| ------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- |
| [`autodocstringstyle`](https://mkdocstrings.github.io/griffe/extensions/official/autodocstringstyle/index.md)             | Set docstring style to `auto` for external packages.                                                                          |
| [`inherited-docstrings`](https://mkdocstrings.github.io/griffe/extensions/official/inherited-docstrings/index.md)         | Inherit docstrings from parent classes.                                                                                       |
| [`public-redundant-aliases`](https://mkdocstrings.github.io/griffe/extensions/official/public-redundant-aliases/index.md) | Mark objects imported with redundant aliases as public.                                                                       |
| [`public-wildcard-imports`](https://mkdocstrings.github.io/griffe/extensions/official/public-wildcard-imports/index.md)   | Mark wildcard imported objects as public.                                                                                     |
| [`pydantic`](https://mkdocstrings.github.io/griffe/extensions/official/pydantic/index.md)                                 | Support for [Pydantic](https://docs.pydantic.dev/latest/) models.                                                             |
| [`runtime-objects`](https://mkdocstrings.github.io/griffe/extensions/official/runtime-objects/index.md)                   | Access runtime objects corresponding to each loaded Griffe object through their `extra` attribute.                            |
| [`sphinx`](https://mkdocstrings.github.io/griffe/extensions/official/sphinx/index.md)                                     | Parse [Sphinx](https://www.sphinx-doc.org/)-comments above attributes (`#:`) as docstrings.                                   |
| [`typing-doc`](https://mkdocstrings.github.io/griffe/extensions/official/typingdoc/index.md)                              | Support for [PEP 727](https://peps.python.org/pep-0727/)'s typing.Doc, "Documentation in Annotated Metadata".                 |
| [`warnings-deprecated`](https://mkdocstrings.github.io/griffe/extensions/official/warnings-deprecated/index.md)           | Support for [PEP 702](https://peps.python.org/pep-0702/)'s warnings.deprecated, "Marking deprecations using the type system". |
